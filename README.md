## Description

Google book search app that uses the google books api to fetch book data when searched. Books can then be save into a mongodb and then rendered on the saved page.

## Technologies Used

- React.js
- Javascript
- Node.js
- Express.js
- Material UI
- Axios
- Mongoose
- MongoDB
- Heroku deployment
